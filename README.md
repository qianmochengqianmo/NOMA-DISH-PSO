This GitHub repository contains the complete simulation framework used in our paper:

“Dynamic Power Allocation in NOMA-based Satellite-Terrestrial Networks: A Customized Particle Swarm Optimization Approach.”

Included Components

DISH-PSO algorithm implementation
(Dirichlet-based initialization, fitness-based sorting, heuristic constraint handling, exponentially decaying inertia weight)

Channel models

Shadowed-Rician fading

Hardware impairment model

CEE and imperfect SIC model

Outage probability evaluation

SINR expressions for multi-user NOMA

Analytical + Monte Carlo simulations

Data generation scripts

Scripts used to generate all figures (Fig. 2–Fig. 8)

Parameter configuration files

Reproducible settings for all experiments

Reproducibility Statement

All figures and numerical results in the manuscript can be fully reproduced using the scripts provided in this repository.
